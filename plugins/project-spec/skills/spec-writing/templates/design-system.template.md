# Design System Template

> Visual design tokens. Create if project has UI needing design consistency.

```markdown
## Design System

> Visual foundation for [Project Name]

## Colors

### Primary

| Token | Value | Usage |
|-------|-------|-------|
| `--color-primary` | #[hex] | Primary actions |
| `--color-primary-hover` | #[hex] | Hover state |

### Neutral

| Token | Value | Usage |
|-------|-------|-------|
| `--color-background` | #[hex] | Page background |
| `--color-surface` | #[hex] | Card backgrounds |
| `--color-text` | #[hex] | Primary text |
| `--color-border` | #[hex] | Borders |

### Semantic

| Token | Value | Usage |
|-------|-------|-------|
| `--color-success` | #[hex] | Success states |
| `--color-error` | #[hex] | Error states |
| `--color-warning` | #[hex] | Warning states |

## Typography

| Token | Value | Usage |
|-------|-------|-------|
| `--font-sans` | '[Font]', sans-serif | Body text |
| `--font-mono` | '[Font]', monospace | Code |

### Sizes

| Token | Size | Usage |
|-------|------|-------|
| `--text-sm` | 14px | Small text |
| `--text-base` | 16px | Body |
| `--text-lg` | 18px | Large |
| `--text-xl` | 24px | Headings |

## Spacing

| Token | Value |
|-------|-------|
| `--space-1` | 4px |
| `--space-2` | 8px |
| `--space-4` | 16px |
| `--space-8` | 32px |

## Radius

| Token | Value | Usage |
|-------|-------|-------|
| `--radius-sm` | 4px | Buttons |
| `--radius-md` | 8px | Cards |
| `--radius-lg` | 12px | Modals |

## Shadows

| Token | Value |
|-------|-------|
| `--shadow-sm` | 0 1px 2px rgba(0,0,0,0.05) |
| `--shadow-md` | 0 4px 6px rgba(0,0,0,0.1) |

---

*Related: Frontend*
```

## When to Create

- Project has custom UI design
- Project needs consistent visual language
- Project has multiple pages/screens

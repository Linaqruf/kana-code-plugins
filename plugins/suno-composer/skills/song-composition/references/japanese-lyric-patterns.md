# Japanese Lyric Patterns

## Syllable Structures

### Traditional Patterns

**5-7-5 (Haiku-inspired):**
```
きみのこえ (ki-mi-no-ko-e) - 5
とおくにきこえる (to-o-ku-ni-ki-ko-e-ru) - 8→7 with elision
やさしいね (ya-sa-shi-i-ne) - 5
```

**7-7 (Tanka-inspired):**
```
なみだがこぼれて (na-mi-da-ga-ko-bo-re-te) - 7
こころがいたいよ (ko-ko-ro-ga-i-ta-i-yo) - 7
```

### Modern Pop Patterns

**4-4-4-4 (Square phrases):**
Common in upbeat pop, easy to sync with beats.

**Free form:**
Most modern J-pop doesn't strictly follow traditional patterns but considers natural speech rhythm.

## Emotional Vocabulary

### Longing/Yearning (切なさ)
- 切ない (setsunai) - painfully emotional, bittersweet
- 恋しい (koishii) - longing for someone/something
- 届かない (todokanai) - can't reach, unreachable
- 遠い (tooi) - distant, far away
- 想い (omoi) - deep feelings, thoughts of someone

### Sadness/Melancholy (悲しみ)
- 悲しい (kanashii) - sad
- 涙 (namida) - tears
- 別れ (wakare) - parting, farewell
- 消えてしまう (kiete shimau) - disappearing, fading
- 忘れられない (wasurerarenai) - unforgettable

### Hope/Light (希望)
- 光 (hikari) - light
- 希望 (kibou) - hope
- 夢 (yume) - dream
- 明日 (ashita) - tomorrow
- 信じる (shinjiru) - to believe

### Beauty/Ephemerality (儚さ)
- 儚い (hakanai) - fleeting, ephemeral
- 美しい (utsukushii) - beautiful
- 桜 (sakura) - cherry blossom (symbol of transience)
- 季節 (kisetsu) - season
- 風 (kaze) - wind

### Love/Affection (愛)
- 愛してる (aishiteru) - I love you
- 好き (suki) - like/love
- 君 (kimi) - you (intimate)
- あなた (anata) - you (formal/romantic)
- 二人 (futari) - the two of us

### Nostalgia (懐かしさ)
- 懐かしい (natsukashii) - nostalgic
- 思い出 (omoide) - memories
- あの日 (ano hi) - that day
- 昔 (mukashi) - the past, long ago
- 変わらない (kawaranai) - unchanged

## Common Phrase Patterns

### Chorus Hooks
```
君に届け (kimi ni todoke) - Reach you
ずっとそばにいて (zutto soba ni ite) - Stay by my side forever
この瞬間を (kono shunkan wo) - This moment
忘れないで (wasurenaide) - Don't forget
```

### Verse Openings
```
あの日から (ano hi kara) - Since that day
気づいたら (kizuitara) - Before I knew it
いつからか (itsu kara ka) - Since when
静かに (shizuka ni) - Quietly
```

### Emotional Builds
```
もう一度 (mou ichido) - Once more
最後の (saigo no) - The last
永遠に (eien ni) - Forever
どこまでも (doko made mo) - Endlessly

```

## Romanization for Suno

### Standard Romanization
Use Hepburn romanization for clarity:
- し = shi (not si)
- ち = chi (not ti)
- つ = tsu (not tu)
- ふ = fu (not hu)
- じ = ji (not zi)

### Long Vowels
Indicate with doubling or macron:
- おおきい = ookii or ōkii
- とうきょう = toukyou or tōkyō

### Particles
Common particles and their romanization:
- は (topic) = wa
- を (object) = wo or o
- へ (direction) = e

## Mixed Language Techniques

### Code-Switching Points
Natural switch points in Japanese-English songs:

**At section breaks:**
```
[Japanese Verse]
[English Chorus]
```

**Within phrases (loan word style):**
```
この feeling 止まらない
(kono feeling tomaranai)
```

**Call and response:**
```
Japanese line → English echo
忘れない → Never forget
```

### Common English Insertions
- Love, heart, dream
- Forever, always
- One more time
- Please, baby
- Start, go, fight

## Rhyme in Japanese

### End Rhyme (less common)
```
hikari (光) / negai (願い)
ashita (明日) / anata (あなた)
```

### Internal Sound Patterns
Focus on vowel repetition:
```
a-i pattern: 愛 (ai) / 会い (ai) / 開い (hirai)
o-u pattern: 空 (sora) / 遠く (tooku)
```

### Rhythmic Repetition
```
どこまでも、どこまでも (doko made mo, doko made mo)
ずっと、ずっと (zutto, zutto)
```

## Line Length Guidelines

### Verse Lines
- Short: 3-5 syllables (intimate, sparse)
- Standard: 7-10 syllables (conversational)
- Long: 12-16 syllables (flowing, narrative)

### Chorus Lines
- Hook: 4-7 syllables (memorable, punchy)
- Extension: 8-12 syllables (development)

### Bridge Lines
Often slightly different rhythm from verse/chorus to create contrast.
